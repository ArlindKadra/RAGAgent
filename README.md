## 🚀 Getting Started



### Prerequisites

- Python 3.12+

- [uv](https://github.com/astral-sh/uv) installed



### Installation



Clone the repository:



```bash

git clone https://github.com/ArlindKadra/RAGAgent.git

cd RAGAgent

# Install dependencies
uv sync
```
